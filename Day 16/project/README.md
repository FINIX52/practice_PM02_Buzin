# День 16. Разработка интеграционных и контрактных тестов

## Описание
Реализация слоя доступа к данным (репозиторий) для управления заказами с интеграционными и контрактными тестами.

## Функциональность
- CRUD операции для заказов и позиций
- Поиск по статусу, диапазону дат
- Подсчёт суммы заказа
- Интеграция с внешним API доставки
