# GUI для таблицы "Товары" (Складской учёт, вариант 4)

## Описание
Приложение позволяет управлять таблицей `Products` базы данных `Variant4_Work`:
- просмотр всех товаров;
- добавление нового товара;
- редактирование выбранного товара;
- удаление товара;
- поиск по наименованию, артикулу или единице измерения;
- экспорт всех данных в CSV.

## Запуск
1. Убедитесь, что MySQL сервер запущен и база данных `Variant4_Work` создана (скрипт из Дня 5).
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
