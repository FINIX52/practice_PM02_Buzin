# Система управления бронированиями (Вариант 4)

## Описание
Система для управления бронированиями отелей с поддержкой групповых бронирований.

## Особенности варианта 4
- Групповое бронирование нескольких номеров
- Единая транзакция (Unit of Work)
- Скидка 5% при бронировании 3+ номеров
- Откат всей транзакции при недоступности любого номера
