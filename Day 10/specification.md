# Спецификация сервиса валидации заказов

## Функция
`validate_order(order: dict) -> dict`

## Бизнес-правила
| № | Правило | Описание |
|---|---------|----------|
| 1 | Сумма заказа | > 0 и < 1_000_000 |
| 2 | Новый пользователь | Если is_new = True → сумма не более 15_000 |
| 3 | Количество позиций | ≤ 50 |
| 4 | Алкоголь | Категория "Alcohol" → age_verified = True, время 08:00–23:00 |
| 5 | Риск-скоринг (сумма) | Если сумма > 100_000 → risk_score = 0.9 |
| 6 | Риск-скоринг (email) | Если email менялся за последний час → risk_score += 0.2 |
| 7 | Риск-скоринг (страна) | Если страна доставки ≠ страна кошелька → risk_score += 0.3 |
}
